# プログラミング実習 課題1：CLIタスク管理ツール「my-todo」

## ■ プロジェクト概要

ターミナルから操作するタスク管理ツールをNode.jsで作成しました。
タスクの追加・一覧表示・ステータス更新・削除をターミナルから行うことができます。

追加したタスクはSQLiteデータベース (todo.db)に保存されます。

## ■ セットアップ手順

### ① リポジトリをクローン

```bash
git clone https://github.com/monmon-gif/my-todo.git
cd my-todo
```

### ② 必要なパッケージをインストール

```bash
npm install
```

---

## ■ 各コマンドの使い方

###  タスクを追加する

```bash
node todo.js add "タスク内容"
```
タスク名を入れることで、タスクを登録することができます。

```bash
node todo.js add "タスク内容（高）" --priority high
node todo.js add "タスク内容（低）" --priority low
node todo.js add "タスク内容（中）"
```
タスク名を入れ、その後に「**high**」か「**low**」を入れることで、優先度を設定することができます。

また、特に指定がなければ優先度は「**medium**」として登録されます。

---

###  タスク一覧を表示する

```bash
node todo.js list
```
タスクの一覧表示します。
```bash
node todo.js list --done
```
完了タスクのみ一覧表示します。
```bash
node todo.js list --todo
```
未完了タスクのみ一覧表示します。

---
###  タスクを完了にする

```bash
node todo.js done "タスクID"
```
タスクIDを入れることで、該当のタスクを完了にすることができます。

---

###  タスクを削除する

```bash
node todo.js delete "タスクID"
```
タスクIDを入れることで、該当のタスクを削除することができます。

### タスクを検索する
```bash
node todo.js search "牛乳"
```
タスク名（部分一致も可）を入れることで、該当のタスクを検索することができます。

### タスクの統計表示をする
```bash
node todo.js stats
```
タスク全体の統計を表示することができます。
また、1週間でのタスク進捗度も表示することができます。
